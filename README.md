# AI Grade Recognition  
**Emotional Intent Analysis from Teacher Reviews**

AI Grade Recognition is a portfolio-focused **Natural Language Processing (NLP)** project that analyzes written student reviews to identify underlying **sentiment and emotional intent** toward teachers. Implemented as a **Jupyter Notebook** using Python, the project demonstrates applied machine learning, transformer-based models, and data-driven insight generation.

The goal is to extract meaningful patterns from unstructured educational feedback and translate them into actionable analytics about teaching quality and student perception.
